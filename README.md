# 洛奇副本组队匹配平台

一个基于 Vercel 和 MongoDB Atlas 的洛奇游戏副本组队匹配平台。

## 功能特性

- 🎮 洛奇副本组队匹配
- 👥 队伍招募和玩家自荐
- 🔍 多种筛选条件
- 💾 MongoDB 数据持久化
- 🚀 Vercel 无服务器部署
- 📱 响应式设计

## 技术栈

- **前端**: HTML5, CSS3, JavaScript, Bootstrap 5
- **后端**: Vercel Serverless Functions, Node.js
- **数据库**: MongoDB Atlas
- **部署**: Vercel
